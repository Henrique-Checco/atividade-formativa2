Sobre

Criamos esse projeto para mostrar os casos de covid existentes no mundo no ano de 2020 utilizando mapas e gráficos com seu casos, suspeitas e mortes.

Nosso objetivo com este projeto é divulgar para a sociedade através de uma API todos os casos, mortes e recuperações que existiram anos atrás sobre o covid.

Utilizamos coisas como o html, css, javascript, biblioteca google charts para a realização deste projeto.
